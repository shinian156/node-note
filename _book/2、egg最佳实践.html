创建项目

# 创建项目 

# egg-init地址： https://github.com/eggjs/egg-init/blob/master/README.zh-CN.md

npm i egg-init -g 

egg-init egg-server --type=simple 

cd egg-server npm i

# 启动项目 
# npm start 和 npm stop 配合使用

npm run dev 

open localhost:7001
